4) 指标口径说明

指标 定义 胜出概率计算方式
自动归类（便于持续加指标）：率值类——指标名含「率」或表内为 D1/D7 留存率时，胜出概率按「成功次数 / 样本人数」的二项模型（双 Beta）比较；成功次数优先取同指标对应的「人数」列（若有），否则取 样本人数 × 该指标率值。ARPU / ARPPU 类——指标名以 ARPU 或 ARPPU 结尾时，与同名称前缀下的「人数 / 率 / 收入」列联动，计算方式与同表 VIP 新购 ARPU / ARPPU 一致（见下）。
VIP新购购买率 VIP新购购买人数 / 样本人数 分子 = 样本人数 × VIP新购购买率；分母 = 样本人数
VIP新购ARPPU VIP新购总收入 / VIP新购购买人数 分母 = 样本人数；转化人数 = VIP新购购买人数；转化收益 = VIP新购总收入
VIP新购ARPU VIP新购总收入 / 样本人数 分母 = 样本人数；转化人数 = VIP新购购买人数；转化收益 = VIP新购总收入
D1/D7留存率 D1/D7留存人数 / 样本人数 分子 = 样本人数 × D1/D7留存率；分母 = 样本人数
发起新聊天D0回复率(样本) 原始表该指标的率值 分子 = 样本人数 × 发起新聊天D0回复率；分母 = 样本人数
发起新聊天率(样本) 原始表该指标的率值 分子 = 样本人数 × 发起新聊天率；分母 = 样本人数
人均主动ABA数(样本) 原始表该指标的人均值 分母 = 样本人数；转化人数 = 达成主动ABA人数；转化收益 = 人均主动ABA数 × 样本人数
人均发起聊天数(样本) 原始表该指标的人均值 分母 = 样本人数；转化人数 = 发起新聊天率 × 样本人数；转化收益 = 人均发起聊天数 × 样本人数
VIP取消订阅率 原始表该指标的率值（或 VIP取消订阅人数 / 样本人数） 分子 = VIP取消订阅人数 或 样本人数 × VIP取消订阅率；分母 = 样本人数
VIP 实验期间新购买-取消订阅率(样本) 原始表该指标的率值（或对应人数 / 样本人数） 分子 = VIP 实验期间新购买-取消订阅人数 或 样本人数 × VIP 实验期间新购买-取消订阅率；分母 = 样本人数
VIP 进组1天内新购买-取消订阅率(样本) 原始表该指标的率值（或对应人数 / 样本人数） 分子 = VIP 进组1天内新购买-取消订阅人数 或 样本人数 × VIP 进组1天内新购买-取消订阅率；分母 = 样本人数
免费试用领取率 原始表该指标的率值（或 免费试用领取人数 / 样本人数） 分子 = 免费试用领取人数 或 样本人数 × 免费试用领取率；分母 = 样本人数
免费试用转订阅ARPU 免费试用转订阅收入 / 样本人数（表内口径） 分母 = 样本人数；转化人数 = 免费试用转订阅人数 或 样本人数 × 免费试用转订阅率；转化收益 = 免费试用转订阅收入
免费试用转订阅率 原始表该指标的率值（或 免费试用转订阅人数 / 样本人数） 分子 = 免费试用转订阅人数 或 样本人数 × 免费试用转订阅率；分母 = 样本人数
VIP续费率 原始表该指标的率值（人数/样本人数） 分子 = 样本人数 × VIP续费率 或对应续费人数；分母 = 样本人数
VIP复购率 原始表该指标的率值（人数/样本人数） 分子 = 样本人数 × VIP复购率 或对应复购人数；分母 = 样本人数
VIP 新购+续费 ARPPU VIP 新购+续费 总收入 / 付费人数（与报表列名一致） 与 VIP 新购 ARPPU 相同：分母 = 样本人数；转化人数 = 同前缀下的购买/付费人数（或 样本人数 × 同前缀购买率）；转化收益 = 同前缀下的总收入
VIP 新购+续费 ARPU VIP 新购+续费 总收入 / 样本人数 与 VIP 新购 ARPU 相同：分母 = 样本人数；转化人数 = 同前缀付费/购买人数（或率×样本）；转化收益 = 同前缀总收入
特定渠道VIP ARPU 该渠道下 VIP收入 / 样本人数 与 VIP 新购 ARPU 相同（按该指标名称前缀匹配同系列的「人数/率/收入」列）
维度VIP来源 VIP ARPU 分来源 VIP收入 / 样本人数 与 VIP 新购 ARPU 相同（前缀与列名一致即可自动联动）
语伴列表用户点击率 原始表率值（或点击人数/样本人数） 分子 = 语伴列表用户点击人数 或 样本人数 × 语伴列表用户点击率；分母 = 样本人数
